# Flux RSS Aggregator

Aplicativo para agregar e ler feeds RSS, incluindo suporte a importação OPML e modo leitor.

## Rodar localmente

**Pré-requisito:** Node.js

1. Instale as dependências:
   `npm install`
2. Configure variáveis de ambiente em `.env.local` (veja `.env.example`)
3. Inicie o app:
   `npm run dev`
